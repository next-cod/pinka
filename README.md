# Pinka – сеть кофеен

Лендинг розовой сети кофеен в Уфе. Демонстрационный проект для портфолио (дизайн + вёрстка).

## Стек

- Один файл `Pinka.dc.html` (Design Context export) + рантайм `support.js` (React 18 UMD, грузится с CDN).
- Чистый HTML/CSS, без сборки — открывается как статика.

## Запуск локально

Нужен HTTP-сервер (из-за подгрузки модулей рантаймом):

```bash
python -m http.server 8099
# затем открыть http://localhost:8099/
```

## Возможности

- Полностью адаптивный дизайн: от 320px до 4K, мобильное меню-гамбургер.
- Изображения оптимизированы в WebP с lazy-loading и приоритетной загрузкой героя.
- Единая дизайн-система: типографика, отступы, ховеры, состояния фокуса.
- Поддержка `prefers-reduced-motion`.

> Кофейня вымышлена: все названия, адреса, контакты и цены придуманы для оформления.
